# Meal Plan Web App Setup
```
docker run --rm i-ti -v $PWD:/usr/src -w /usr/src ruby:2.3.1 bash
gem install rails
rails new meal_plan --skip-turbolinks --skip-spring -d postgresql --skip-bundle --skip-test

```
